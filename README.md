## PyCraft

Early Minecraft Classic–inspired 2D sandbox game.

### Features
- 2D Terraria-style world
- Grass / Dirt / Cobblestone layers
- Block breaking & placing
- Player movement + gravity
- Minecraft-style hotbar
- ESC to return to menu

### Controls
- A / D – Move
- Space – Jump
- Left Click – Break block
- Right Click – Place block
- 1 / 2 / 3 – Select block
- ESC – Return to menu
